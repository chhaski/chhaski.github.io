Index

as knowledge infrastructure, cross-reference mechanism, way of containing knowledge
an index can operate like a hyperlink, creating associative relations between things



    Methodology

    pre-processing corpus.
    make each index structure uniform parse, reassemble
    processing corpus.
    create vectors for each sequence of text (subentry) in the corpus reduce dimensionality of each vector with t-SNE save the x-values of t-SNE visualization of vectors in 2-d space as an array, map it to a better range create dictionary of each subentry and its corresponding vector
    generating.
    with tracery grammar the position of each subentry on the page is determined by the x-position of the vectors in 2-d space as determined by t-SNE

Michel Foucault - 'History of Sexuality Volume 1', José Esteban Muñoz - 'Cruising Utopia', Sara Ahmed - 'Queer Phenomenology', some phrases from my notebook.
